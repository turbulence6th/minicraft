MiniCraft Tutorial

Welcome to MiniCraft Browser Game

You can move, acquire an item and damage with direction keys.

You can eat with the key "E"

You can click the item on the list "Click to merge" to merge an item

You can click on the item in the inventory to use an item if it is usable

You can put an item with the keys "W", "A", "S" and "D" to relative direction.

You can save and quit with the key "Q"

Have fun
